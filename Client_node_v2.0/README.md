WHOIS API Service
==================
a small tools for query domain whois

### USE
```curl http://ip:prot/WHOIS/<domain>```


### Feature

* logging
* JSON-setting
* using flask
* FQDN -> domain
* base the RCF 3912
* connect the WHOIS service
* analysis WHOIS extract character
